Cat Face Identification

A classical computer vision system for cat face detection using handcrafted HOG features and Support Vector Machine (SVM) classification, demonstrating accurate and interpretable image recognition without deep learning.

Project Overview

This project presents a lightweight computer vision pipeline for identifying cat faces using traditional image processing and machine learning techniques. Facial landmarks are used for localization, Histogram of Oriented Gradients (HOG) extracts discriminative features, and a Linear Support Vector Machine (SVM) performs binary classification. The approach provides high accuracy while remaining computationally efficient and fully explainable.
