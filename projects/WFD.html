Edge AI | Real-Time Face, Age & Gender Detection

Edge AI

Real-time Face Detection, Age Estimation, and Gender Classification system using OpenCV DNN models. The application performs on-device inference through a webcam, enabling low-latency, privacy-preserving, and offline AI.

Project Overview

This Edge AI application performs real-time face detection, age prediction, and gender classification directly on an edge device without requiring cloud connectivity. Built using OpenCV's Deep Neural Network (DNN) module and pre-trained Caffe models, the system delivers fast, privacy-focused inference suitable for computer vision, IoT, and embedded AI applications.
