Agentic Async Federated Learning

An AI-driven Asynchronous Federated Learning framework that enhances communication efficiency, scalability, and privacy for heterogeneous edge-cloud environments using intelligent agents and adaptive aggregation.

Project Overview

This research introduces an Agentic AI-integrated Asynchronous Federated Learning (AsyncFL) framework that enables efficient collaborative model training without sharing raw data. Intelligent edge and cloud agents dynamically evaluate update quality, network conditions, resource availability, and model freshness to optimize communication and aggregation while maintaining privacy and model performance.
